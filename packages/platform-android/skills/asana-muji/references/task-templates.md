# Asana Task Templates

Templates for common Android MUJI task types. Used by `create-task.md` workflow.

## Feature Task

| Field | Value |
|-------|-------|
| Name | `[Component] Feature: {description}` |
| Project | Android Libraries |
| Section | Backlog |
| Tags | `feature`, `android` |
| Description | What it does, acceptance criteria, design reference |

## Bug Task

| Field | Value |
|-------|-------|
| Name | `[Component] Bug: {description}` |
| Project | Android Libraries |
| Section | Bugs |
| Tags | `bug`, `android` |
| Description | Steps to reproduce, expected vs actual behavior, Android version, device |

## Design Token Update

| Field | Value |
|-------|-------|
| Name | `[Tokens] Update: {token name}` |
| Project | Android Libraries |
| Section | Design Tokens |
| Tags | `tokens`, `android` |
| Description | Token path, old value, new value, affected Composables |
