# Asana Task Templates

Templates for common iOS MUJI task types. Used by `create-task.md` workflow.

## Feature Task

| Field | Value |
|-------|-------|
| Name | `[Component] Feature: {description}` |
| Project | iOS Libraries |
| Section | Backlog |
| Tags | `feature`, `ios` |
| Description | What it does, acceptance criteria, design reference |

## Bug Task

| Field | Value |
|-------|-------|
| Name | `[Component] Bug: {description}` |
| Project | iOS Libraries |
| Section | Bugs |
| Tags | `bug`, `ios` |
| Description | Steps to reproduce, expected vs actual behavior, iOS version |

## Design Token Update

| Field | Value |
|-------|-------|
| Name | `[Tokens] Update: {token name}` |
| Project | iOS Libraries |
| Section | Design Tokens |
| Tags | `tokens`, `ios` |
| Description | Token path, old value, new value, affected components |
