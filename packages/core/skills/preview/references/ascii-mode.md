# --ascii mode

Generate terminal-friendly ASCII diagram only (no Mermaid, no HTML).

## Rules
- Box-drawing chars: ┌─┐│└┘├┤┬┴┼
- Max 60 columns wide
- Arrows: → ← ↑ ↓ ↔ ──▶ ◀──
- Show hierarchy, flow, or relationships
- Add a 1-line legend if symbols need clarification
- Wrap in a code block (```text)
