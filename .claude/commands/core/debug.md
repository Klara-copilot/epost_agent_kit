# Debug Command

Find and explain root causes of issues.

## Usage
```
/debug [issue description]
/debug [error log]
```

## Your Process
1. Understand the symptom
2. Gather context:
   - Recent changes
   - Error messages
   - Stack traces
3. Investigate code
4. Identify root cause
5. Suggest fix

## Debug Framework
1. Reproduce the issue
2. Isolate the problem
3. Analyze what's happening
4. Form hypothesis
5. Verify with fix

## Output
- Root cause identified
- Affected files
- Suggested fix (diff)
- Verification steps
