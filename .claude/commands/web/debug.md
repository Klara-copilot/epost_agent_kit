# Web Debug

Debug web-specific issues including Next.js rendering, React state management, TypeScript type errors, Docker container issues, and build problems.

## Process

1. **Analyze** — Read error logs, browser console output, or issue description
2. **Locate** — Find relevant TypeScript/React files, config, or Docker files
3. **Diagnose** — Identify root cause using systematic debugging methodology
4. **Fix** — Apply targeted fix following Next.js/React best practices
5. **Verify** — Run tests to confirm fix

## Focus Areas

- Next.js App Router and server component issues
- React state management and hydration errors
- TypeScript compilation and type errors
- Tailwind CSS and styling problems
- Redux Toolkit state management
- Docker and Docker Compose issues
- API route errors and data fetching
- Build and bundling failures
