# Design Fast Command

Quickly implement simple UI components and designs.

## Usage

```
/design:fast [description of UI needed]
```

## Examples

- `/design:fast Create a simple button component`
- `/design:fast Add a loading spinner`
- `/design:fast Make this card responsive`

## Best For

- Simple components
- Quick layout fixes
- Basic styling updates
- Minor UI tweaks

## Process

1. Understand requirements
2. Create/update component
3. Apply appropriate styling
4. Ensure responsiveness
5. Check accessibility

## Output

- UI component implementation
- Styling (CSS/Tailwind/etc.)
- Responsive layout
